How to Add New YouTube Videos

Keep adding unlimited YouTube links by updating one simple file: videos.js

1

Copy YouTube Link

Example: https://youtu.be/mbPNkDEN3Ps

2

Take Video ID

For this link, video ID is mbPNkDEN3Ps

3

Add in videos.js

Paste the ID, title, age, category, and description.

{
  title: "Kids Learning Video",
  youtubeId: "mbPNkDEN3Ps",
  age: "2-4",
  category: "Rhymes",
  audience: "Both",
  description: "Educational video for kids."
}
